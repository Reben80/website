How to Play:

  1. Select a color from the palette below the graph.
  2. Click on a vertex to color it.
  3. Adjacent vertices cannot have the same color.
  4. Try to color the entire graph using as few colors as possible.
  5. Use the "Reset" button to start over or generate a new random graph.
  6. Select a specific challenge from the dropdown menu to try different graph structures.
